# Política de Privacidad – Gestos de Gastos

**Última actualización:** 25/12/2025

**Responsable:** Persona Física  
**Contacto:** ndirenzo03@gmail.com  

## 1. Datos que recopilamos
La aplicación **Gestos de Gastos** no solicita ni recopila datos personales identificables (nombre, correo, DNI, etc.).

## 2. Datos almacenados
Los datos de gastos y presupuestos se guardan únicamente en el dispositivo del usuario. No se envían ni se almacenan en servidores externos.

## 3. Publicidad
La aplicación utiliza Google AdMob para mostrar anuncios. AdMob puede recopilar identificadores del dispositivo y datos de uso con fines publicitarios.  
Más info: https://policies.google.com/privacy

## 4. Permisos
La aplicación puede solicitar permiso de cámara (`android.permission.CAMERA`) para escanear tickets de compra.

## 5. Compartición de datos
No compartimos datos personales con terceros. Solo los servicios publicitarios (AdMob) pueden recopilar información técnica según sus propias políticas.

## 6. Seguridad
La información se guarda localmente en el dispositivo. No realizamos transmisión de datos a servidores.

## 7. Cambios en esta política
Cualquier cambio en esta política será publicado en este mismo lugar con la fecha correspondiente.
